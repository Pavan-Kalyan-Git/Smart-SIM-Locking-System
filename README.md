# Smart SIM Locking System

## Overview
A hardware-based security system that prevents unauthorized SIM card removal using biometric authentication.

## Hardware Used
- Raspberry Pi 4
- R307S Fingerprint Sensor
- 28BYJ-48 Stepper Motor
- ULN2003 Motor Driver

## Software Used
- Python
- Tkinter GUI
- SQLite Database
- RPi.GPIO

## Features
- Fingerprint authentication
- Password backup system
- Physical SIM tray locking
- Real-time protection

## Project Files
- smart_sim_lock.py → Real hardware code
- demo_sim_lock.py → Demo version (no hardware)

##  Note
The real system requires Raspberry Pi hardware.
Use demo version for simulation.
